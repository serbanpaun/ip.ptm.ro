# Privacy Notice

This project includes an index.php page that displays information such as the visitor’s IP address, browser details, operating system, and general system metadata. This notice explains what data is processed and why.

## What Data Is Collected
When a visitor accesses the webpage, the following information may be displayed or logged:

- IP address
- Browser and user-agent details
- Operating system
- Device type
- Referrer URL (if available)
- Date and time of access
- General server environment variables (depending on configuration)

## Purpose of Data Processing
This information is collected and displayed strictly for:
- Diagnostics and debugging
- Security monitoring (e.g., identifying suspicious activity)
- General analytics and statistics
- Demonstration of server-side PHP functionality
- No profiling or marketing use is performed.

## Legal Basis (GDPR)
If you are subject to GDPR, processing is based on:
- Legitimate interest (Art. 6(1)(f) GDPR): ensuring website security, functionality, and diagnostics.

## Data Storage
By default, the project does not store visitor data permanently.
If logging is enabled on your server, logs may be retained according to your server’s rotation policy (typically 1–30 days).
You may modify the project to store data, but then you must update this notice accordingly.

## Data Sharing
Visitor data is not shared with third parties, except:
- Hosting provider log retention required for normal server operation.
- Google Analytics or other tools if you explicitly enable them in the project.

## Third-Party Services (if enabled)
If you use Google Analytics or similar services, they may collect additional data according to their own policies.
Add your own Google Analytics ID to the script included in the project.

## User Rights
If GDPR applies, users may request:
- Access to their personal data
- Correction or deletion (where applicable)
- Information on how their data is processed
